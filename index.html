Tribo-electric Nano Generator

A mat with TENG sensors to identify a person by their unique walking pattern. Used an esp32 MCU with C++ and BLE to communicate. Nanoprocessor

4-bit nano processor which can run basic arithmetic (addition, subtraction, left shift, right shift, and negation). I contributed by creating all the subcomponents in VHDL, simulating, and debugging. The main components were 4-bit Add/Subtract unit, 3-bit adder, 3-bit Program Counter (PC), 2-way 4-bit multiplexers, Register Bank, Program ROM, Buses, Instruction Decoder.
2023

Medi-box with esp32 and NodeRED

I Created a medibox for Embedded Systems and Applications module with esp32 and made a NodeRED dashboard. Feature of the medibox are Setting/Disabling alarms, setting time zone, Automatic time fetching from NTP server, MQTT communication with NodeRED, and monitoring temperature and humidity with dht22 sensor.
